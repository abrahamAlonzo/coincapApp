# Getting Started

## Requeriments: 

 - Android studio
 - xcode
 - nodejs: 12.0



```Shell
git clone https://github.com/abrahamAlonzo/coincapApp.git
cd coincapApp/CoinCapApp
yarn install
cd ios
pod install
cd ..
react-native run-ios
```
